Comandos Git a usar:

git init	## Le dice a la computadora que vamos a usar git en este repositorio
git status	## Ver en que estado estan los archivos
git add <file>	## Añade el archivo a la "staging area" se guarda pero no se agrega todavia al repositorio
git config --global user.email "you@example.com" ##Para Identificarte en Github
git config --global user.name "Your Name"	 ##Para Identificarte en Github paso subsiguiente
git commit -m "Primer commit para guardar archivos explicativos a mis compañeros" ##Esto es un commit guarda todo el trabajo realizado y esta listo para subirse a github
git push	##Sube todos los cambios a Github. ATENCION!! te va a pedir usuario y contraseña, en usuario pone tu nombre de usuario pero en la contraseña pone tu token
git status ##Ver el estado de los archivos creados o modificados
git pull  ##Recibe todos los cambios que se hicieron en el archivo por tus compañeros
